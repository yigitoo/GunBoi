# SnakeGame
A space invaders game prepared by Ruby2D.

To install Ruby2D lib from Gemfile in repository:
<br>
```bash
$ bundle install
```
or just run:
```bash
$ gem install ruby2d
```
After installation, <br>
To run codes i made:
<br>
```bash
$ make
```
